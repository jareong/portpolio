내가 생각하는 사용자경험 - User Experience

UX의 정의에 대해서 알아보고, 다양한 UX의 사례들, 특히나 성공적인 UX라 불리는 사례들을 보면서
사용자 경험을 바탕으로 어떤 제품이나 서비스를 구축하는 것의 중요성을 깨달았다. 내가 생각하는 UX란, 사용자가 원하는 기능을 사용할 때 사용자가 불편함을 느끼지 않고 사용할 수 있도록 하고,
원하는 활동을 편리하게 제품이나 서비스를 이용하여 할 수 있도록 하는 것이라는 생각이 들었다.
사전적인 의미에서 UX란 사람의 여러 감각과 감정의 총합을 뜻하는데, 내가 생각하는 UX는 ‘불편함’이라는,
혹은 다르게 불릴 수도 있는 부정적인 감정을 최소화시키는 것이라는 생각이 든다.

질문을 선택하시면 정리된 답변을 보실수 있습니다.

1. html5의 가장 큰 특징은 무엇일까요? 화살표
1. 상업적, 독점적 플러그인을 대체할 수 있는 웹 표준 기술로 다양한 플랫폼을 동시에 지원가능
2. 별도의 플러그인 없이 동영상 재생 및 Flash 기능과 javascript 기능을 지원함
3. Device, OS, Browser 등 서로다른 환경에서 접근하는 이용자들에게 별도의 프로그램 추가 설치 없이 다양한 정보를 동일하게 보여 줄 수 있음
2. 웹표준, 웹접근성, 시멘틱웹에 대해 설명해 주세요. 화살표
*웹표준 : 웹표준화 단체(W3C)가 권고한 표준안을 말하며, 용도에 맞는 태그 사용과 CSS를 분리하여 작업하는 것
*웹접근성 : 웹 사이트에서 제공하는 정보를 사람의 신체 조건 및 제한 없이 모두가 동등하게 이용할 수 있도록 보장하는 것
*시멘틱웹 : 개발자가 의도한 요소의 의미가 명확히 드러나게 작성하는 방법이자, 방식. 코드의 가독성을 높이며 유지보수를 높일 수 있다.
3. 자바스크립트 라이브러리 사용시 가장 큰 장점은? 화살표
라이브러리란 특정한 코드 (함수 혹은 클래스)를 포함하고 있는 컴파일된 파일이며, 이렇게 자주 쓰이는 함수들을 라이브러리에 모아 사용시 프로그램 유지, 디버깅이 수월하며 컴파일 시간을 단축 시킬 수 있다.
4. .setInterval() 함수에 대해 설명하시오. 화살표
일정한 시간 간격을 두고 반복해서 실행하고 싶을 때 사용합니다. (예: 달리는 아이 / 이건 제 포폴에 넣을 예정이라..)
5. CSS 미디어쿼리는 어떤 역할을 수행하는지 답하시오~! 화살표
미디어쿼리는 사이트에 접속하는 장치(브라우저, 휴대폰, 패드)에 따라 특정한 CSS 스타일을 사용하도록 도와주는 소프트웨어 모듈입니다.
즉. 화면에 맞게 레이아웃을 변경해 주는 것입니다.
